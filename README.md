# Nova (NOVA) Smart Contracts

This repository contains the open-source smart contract code for the NOVA token, which is deployed on the BNB Smart Chain (BSC).

The purpose of this repository is to provide transparency, verifiability, and technical reference for developers, auditors, and community members.

---

## Overview

- **Project Name:** Nova
- **Token Symbol:** NOVA
- **Blockchain:** BNB Smart Chain (BSC)
- **Contract Type:** ERC20-compatible utility token
- **Status:** Deployed & Verified

NOVA is designed as a utility token to support on-chain operations, network incentives, and ecosystem participation within the Nova project.
